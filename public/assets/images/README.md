# Diretório de Imagens

Este diretório é destinado às imagens do projeto.

## Como usar

Coloque suas imagens aqui e elas estarão disponíveis em:
- `/assets/images/nome-da-imagem.jpg`

## Estrutura Sugerida

```
images/
├── logo/
│   └── logo.svg
├── artists/
│   ├── artist-1.jpg
│   ├── artist-2.jpg
│   └── ...
└── portfolio/
    ├── portfolio-1.jpg
    ├── portfolio-2.jpg
    └── ...
```

As imagens atualmente estão sendo carregadas do Unsplash como placeholders.
Substitua as URLs no componente `home.component.ts` pelos caminhos locais quando tiver as imagens reais.
